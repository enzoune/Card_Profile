# 🚀 Cartão de Perfil — Projeto HTML & CSS

## 📌 Sobre o Projeto

Este projeto consiste em um Cartão de Perfil moderno, desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de HTML5 e CSS3.

A proposta foi criar uma interface visualmente atraente, simulando um card de perfil profissional com informações, redes sociais e métricas de interação — algo comum em aplicações reais como redes sociais e dashboards.

<br>

## 🎯 Objetivos
- Praticar estruturação semântica em HTML
- Aprimorar habilidades de estilização com CSS
- Trabalhar com:
  - Flexbox
  - Pseudo-elementos (::before)
  - Organização de layout
 
<br>

## 🖥️ Preview do Projeto

Um cartão contendo:

- 📷 Foto de perfil
- 👤 Nome e profissão
- 🔗 Links para redes sociais
- 📩 Botões de ação
- 📊 Indicadores de engajamento (likes, comentários e compartilhamentos)

<br>

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Google Fonts (Poppins)
- Boxicons (ícones)
